I can recommend postnuke over php nuke. There is a whole community around post nuke, and a lot more features. They have a new version (7.5) you can get at http://www.postnuke.org ... The direct link to the download is here :

http://download.postnuke.com/pafiledb.php?action=category&id=45


They have read me instructions on the install but its pretty easy. You need a server with php4 and mysql running on it. If you have a plesk or cpannel app you can easily create a default database and a user. After you create the database its as easy as 1,2,3

(1)You ftp the files to you server under /myserver/httpdocs
(2)make sure the config files are readable and write-able
/myserver/httpdocs/config.php and /myserver/httpdocs/config.php.old


(3)then you open a browser and type http://www.myserver.com/install.php

The install.php will walk you through a wizard like interface to configure the rest.
